运行  
1.启动本地socket服务器 server/start.bat  
2.安装http服务器  
--npm install http-server -g  
--build\web-mobile\目录下执行命令行 http-server  
--http启动 复制http服务器地址  
3.浏览器打开 http服务器地址 + "?server=xxx"  
4.?server=xxx 的xxx替换上socket服务器地址 http://本机ip +":3000"
